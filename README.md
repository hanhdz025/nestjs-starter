## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development mode
$ pnpm dev

# production mode
$ pnpm build
$ node dist/main.mjs
```

## Build & run with docker

```bash
$ docker build -t nestjs-starter .
$ docker run -dp 3000:3000 nestjs-starter
```

## With docker-compose

```bash
$ docker-compose up --build
```
